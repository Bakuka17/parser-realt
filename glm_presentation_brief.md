# ТЗ для GLM — презентация коммерческой недвижимости (вставлять в z.ai при заказе)

Сделай премиум-презентацию объекта недвижимости в PowerPoint (формат .pptx), 16:9, 8–9 слайдов.
Данные объекта и фото — ниже/прикреплены. Используй ТОЛЬКО их, ничего не выдумывай.

## ⚠️ 5 ПРАВИЛ (в прошлый раз ты их нарушил — теперь НЕ нарушай)

1. **ФОТО** — все приложенные, как есть, чётко и целиком, встрой прямо в слайды.
   НЕ генерируй свои картинки, НЕ ставь заглушки, не пересжимай до артефактов.
2. **ШРИФТ С КИРИЛЛИЦЕЙ** — весь текст русский. Заголовки Georgia, текст Arial (есть на любом
   компьютере, с русскими буквами). НИКАКОГО Carlito / Calibri-клонов — прошлый текст ИСЧЕЗ
   из-за этого. Встрой шрифты в файл (Save → Embed fonts), чтобы не билось на чужом компьютере.
3. **НЕ ОБРЕЗАТЬ текст** — каждое слово помещается в карточку/слайд, блоки тянутся/ужимаются,
   ничего не уезжает за край. Единицы целиком: «217 м²».
4. **НИКАКИХ ЗАГЛУШЕК** — без «+375 (ХХ)…», «[Имя Фамилия]», lorem ipsum. Только реальные
   данные; чего нет — блок просто опусти.
5. **КАРТОЧКИ ЕДИНООБРАЗНЫ** — карточки одного блока одинаковы: размер, паддинг, выравнивание.

## СТРУКТУРА (8–9 слайдов)
1. **Титул** — название, подзаголовок, «адрес · метро · цена», одно крупное фото.
2. **Локация** — текст о расположении/трафике + фото или карта.
3. **Параметры** — 6–8 единообразных карточек с числами (площадь, этажи, входы, эл. мощность,
   парковка, режим, цена за м²). Только факты.
4. **Планировка** — планы этажей (если есть) + 3–4 пункта о зонировании.
5–6. **Преимущества** — карточки с ВЫГОДОЙ. НЕ повторяй факты из «Параметров» — давай новую
   ценность (трафик, видимость, гибкость, окупаемость).
7. **Галерея** — все фото сеткой 3 в ряд.
8. **Условия + форматы** — ставка/итог в месяц + подходящие форматы бизнеса.
9. **Контакт** — крупно реальный телефон, имя агента, адрес (нет данных — слайд опусти).

## ТЕКСТ
Конкретно, без воды и канцелярита. Каждый факт — ОДИН раз (без повторов).

## ДИЗАЙН
Премиум: тёмная тема (#0d0d0f) или светлая; один акцент (золото/нейтраль); высокий контраст, воздух.

## ВЫВОД
Один файл .pptx: фото и шрифты встроены, открывается офлайн на любом компьютере.
