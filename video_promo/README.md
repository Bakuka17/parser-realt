# Промо-ролики объектов (Remotion)

Видео-карточка объекта из нашей базы: фото с наездом, счётчик цены, адрес/площадь,
кнопка телефона. Чёрно-золотая тема дашборда. 1080×1350 (вертикаль под соцсети), 7 сек.

## Запуск

```bash
cd ~/realty_env/video_promo
npx remotion studio            # живой предпросмотр в браузере
npm run render                 # рендер → out/promo.mp4
```

Зависимости уже установлены (`node_modules`, ~520 МБ, в git не хранятся —
при чистом клоне: `npm install`).

## Данные объекта

Правятся в `promoDefaults` (`src/Promo.tsx`): тип, адрес, площадь, цена, телефон,
источник + фото `public/photo.png`. Фото берём из кэша дашборда
`web/photos_cache/{хэш}.png`, поля — из строки `commercial_realty.xlsx`.

Демо собрано на живом объекте: Торговое 427 м², $555 100, Сенницкий с/с 68, megapolis.
